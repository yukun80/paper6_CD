# 工作日志（2026-03-11）

## 1. 本次目标修订
- 回顾 `PPO-main` 与早期 `sar_prompt_flood` 的差异，确认 DINO 与原始 Q-learning 不是核心可迁移价值。
- 将 `sar_prompt_flood` 从“GF3_Henan-only 无监督原型”重构为“有标注参考集训练 + GF3_Henan 无标签零样本迁移”的监督参考实验分支。
- 删除原有 GF3 纯无监督入口和配置，统一收敛到 reference-guided prompt proposal + prompt policy + `SAM vit_b` 的新链路。

## 2. 数据与任务重新定义

### 2.1 数据角色
- 参考训练集：
  - `datasets/urban_sar_floods_test/urban_sar_floods_test_tiles_512_band5_band7`
  - 结构：`pre/`, `post/`, `GT/`
  - 标签口径：二分类洪水 `0/1`
- 目标推理集：
  - `datasets/GF3_Henan/GF3_Henan_tiles_512`
  - 结构：`pre/`, `post/`
  - 当前无标签，仅做零样本迁移推理

### 2.2 当前任务定义
- 训练阶段：
  - 在参考训练集上训练 reference-guided prompt proposal
  - 在参考训练集上训练 prompt policy scorer
- 推理阶段：
  - 对每个 GF3 target tile 从参考库检索单个最相近样本
  - 生成初始 prompt，进行 prompt 优化，再送入 `SAM vit_b`
  - 输出 tile 级预测与无监督统计

## 3. `sar_prompt_flood` 重构内容

### 3.1 删除旧 GF3 无监督链路
- 已删除模块：
  - `sar_prompt_flood/config.py`
  - `sar_prompt_flood/config/gf3_henan.json`
  - `sar_prompt_flood/data.py`
  - `sar_prompt_flood/gf3_preprocess.py`
  - `sar_prompt_flood/infer.py`
  - `sar_prompt_flood/run_gf3_pipeline.py`
- 删除原因：
  - 原有链路绑定 `GF3_Henan-only + 纯无监督 change prior`
  - 与当前“参考训练集监督学习 + GF3 零样本迁移”目标不一致

### 3.2 保留并中性化的公共模块
- `sar_prompt_flood/segmenter.py`
  - 保留纯 `SAM vit_b` 后端
- `sar_prompt_flood/optimizer.py`
  - 保留 prompt 子集优化环境与规则贪心优化器
- `sar_prompt_flood/prompts.py`
  - 去掉旧 `PromptCandidateGenerator`
  - 只保留 `CandidateNode`、`PromptSet` 及通用候选点工具函数
- 新增 `sar_prompt_flood/feature_utils.py`
  - 承接旧 `gf3_preprocess.py` 中仍需复用的 `robust_unit / safe_log_ratio`

### 3.3 新增监督参考训练子系统
- 新增配置：
  - `sar_prompt_flood/reference_config.py`
  - `sar_prompt_flood/config/urban_sar_reference.json`
- 新增数据读取：
  - `sar_prompt_flood/reference_data.py`
  - `UrbanSARReferenceTileDataset`：读取参考训练集 `pre/post/GT`
  - `GF3TargetTileDataset`：读取目标推理集 `pre/post`
  - 若参考集 `splits/` 不存在，自动生成 `train.txt / val.txt / ref_bank.txt`
- 新增模型与训练：
  - `sar_prompt_flood/reference_models.py`
  - `sar_prompt_flood/train_reference_prompt.py`
  - 在参考训练集上学习 reference-guided prompt proposal
- 新增 policy 学习：
  - `sar_prompt_flood/reference_policy.py`
  - `sar_prompt_flood/train_prompt_policy.py`
  - 在参考训练集上基于动作增益监督训练 prompt action scorer
- 新增监督版 prompt 生成：
  - `sar_prompt_flood/supervised_prompts.py`
  - 用参考样本 prototype 与 query 特征生成正负候选点

### 3.4 新增 GF3 目标域零样本推理入口
- 新增入口：
  - `sar_prompt_flood/run_reference_prompt_pipeline.py`
- 当前默认行为：
  - 从 `reference_data.ref_bank_split` 中检索单个参考样本
  - 对 `target_data.root` 下每个 GF3 tile 生成 `init_mask -> opt_mask`
  - 输出：
    - `predictions/*.png`
    - `predictions/*_mask.tif`
    - `visuals/*.png`
    - `summary.json`
    - `tile_metrics.json`
- 评估口径：
  - 若输入样本含 `GT`，输出 `IoU/Dice`
  - GF3 目标集无标签时，只输出无监督统计：
    - `mean_init_mask_area_ratio`
    - `mean_opt_mask_area_ratio`
    - `mean_num_pos / mean_num_neg`
    - `low_confidence_tiles`
    - `reference_selection_top10`

## 4. 当前默认配置与运行方式

### 4.1 默认配置
- 参考训练集：
  - `reference_data.root = datasets/urban_sar_floods_test/urban_sar_floods_test_tiles_512_band5_band7`
- 目标推理集：
  - `target_data.root = datasets/GF3_Henan/GF3_Henan_tiles_512`
- 默认配置文件：
  - `sar_prompt_flood/config/urban_sar_reference.json`

### 4.2 运行方式
```bash
# 下载 SAM 权重
python datasets/script/download_sam_vit_b.py

# 训练 reference-guided prompt proposal
python -m sar_prompt_flood.train_reference_prompt \
  --config-file sar_prompt_flood/config/urban_sar_reference.json

# 训练 prompt policy
python -m sar_prompt_flood.train_prompt_policy \
  --config-file sar_prompt_flood/config/urban_sar_reference.json

# 在 GF3_Henan_tiles_512 上做零样本推理
python -m sar_prompt_flood.run_reference_prompt_pipeline \
  --config-file sar_prompt_flood/config/urban_sar_reference.json
```
